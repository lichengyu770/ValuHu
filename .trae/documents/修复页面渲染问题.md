## 修复页面空白/显示源代码问题

### 问题分析
用户看到 HTML 源代码而不是渲染的页面，这说明：
1. 浏览器可能没有正确加载 JavaScript
2. 可能有缓存问题
3. 或者 Python http.server 的 MIME 类型配置问题

### 解决方案步骤

#### 步骤 1：强制刷新浏览器
- Windows: 按 `Ctrl + F5`（强制刷新，不使用缓存）
- 或者按 `Ctrl + Shift + Del` 清除缓存后按 `F5` 刷新

#### 步骤 2：使用 Vite 开发服务器（推荐）
使用 `npx vite --port 3005` 启动，而不是 Python http.server，这样可以确保：
- 正确的 MIME 类型
- 热更新支持
- 更好的开发体验

#### 步骤 3：检查浏览器控制台错误
- 按 `F12` 打开开发者工具
- 查看 Console 标签页是否有红色错误

### 预期结果
用户应该能看到：
- 页面标题："欢迎使用房产估价核心引擎"
- 三个功能卡片（房产估价、高级搜索、报告生成）
- 底部"为什么选择我们"区域

### 优先级
高优先级：确保页面正确渲染
中优先级：解决可能的缓存问题
低优先级：优化服务器配置
