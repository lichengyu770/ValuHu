# ValuHub平台使用指南

本指南将帮助您使用ValuHub平台的各项功能。

## 1. 平台概述

ValuHub是一个房产价值生态引擎，提供专业的房产估价服务，服务大众、政务、企业、院校等多个终端，构建完整的房产价值生态系统。

## 2. 核心功能

### 2.1 企业端功能

#### 2.1.1 CSV/Excel上传
- **功能说明**：支持批量上传房产数据进行估价
- **使用方法**：
  1. 登录企业端
  2. 进入"批量估价"页面
  3. 选择CSV/Excel文件
  4. 点击"上传"按钮
  5. 等待处理完成，下载结果

#### 2.1.2 资产看板
- **功能说明**：提供不动产组合总览、价值波动图表和风险预警
- **使用方法**：
  1. 登录企业端
  2. 进入"资产看板"页面
  3. 查看资产总览、价值趋势和风险预警
  4. 点击具体资产查看详情

#### 2.1.3 PDF估价报告
- **功能说明**：自动生成符合金融机构格式的PDF估价报告
- **使用方法**：
  1. 登录企业端
  2. 进入"估价记录"页面
  3. 选择需要生成报告的估价记录
  4. 点击"生成报告"按钮
  5. 等待生成完成，下载PDF报告

### 2.2 数据后台功能

#### 2.2.1 用户与用量监控
- **功能说明**：查看各端用户增长、API调用次数和活跃度
- **使用方法**：
  1. 登录数据后台
  2. 进入"监控中心"页面
  3. 查看用户增长趋势
  4. 查看API调用统计
  5. 查看活跃度分析

#### 2.2.2 数据质量管理
- **功能说明**：管理数据质量，包括低置信度估价检测、异常数据标记和手动修正
- **使用方法**：
  1. 登录数据后台
  2. 进入"数据质量"页面
  3. 查看低置信度估价列表
  4. 标记异常数据
  5. 手动修正估价结果
  6. 查看模型精度分析报告

#### 2.2.3 运营工具
- **功能说明**：发送系统通知和管理官网博客内容
- **使用方法**：
  1. 登录数据后台
  2. 进入"运营工具"页面
  3. 发送系统通知
  4. 管理博客内容

### 2.3 官网功能

#### 2.3.1 开发者中心
- **功能说明**：展示API文档、SDK和用例
- **使用方法**：
  1. 访问官网
  2. 点击"开发者中心"
  3. 查看API文档
  4. 下载SDK
  5. 查看使用用例

#### 2.3.2 合作伙伴展示墙
- **功能说明**：展示早期合作企业、院校、协会
- **使用方法**：
  1. 访问官网
  2. 点击"合作伙伴"
  3. 查看合作伙伴列表
  4. 点击合作伙伴查看详情

#### 2.3.3 数据洞察博客
- **功能说明**：定期发布市场报告和数据分析
- **使用方法**：
  1. 访问官网
  2. 点击"数据洞察"
  3. 浏览博客文章
  4. 搜索特定主题
  5. 订阅博客

## 3. API使用

### 3.1 API密钥获取

1. 登录企业端
2. 进入"API管理"页面
3. 生成API密钥
4. 复制API密钥

### 3.2 API调用示例

```bash
curl -X POST https://your-domain.com/api/valuations -H "Content-Type: application/json" -H "Authorization: Bearer YOUR_API_KEY" -d '{"property": {"address": "北京市朝阳区...", "city": "北京", "district": "朝阳区", "area": 100, "bedrooms": 2, "bathrooms": 1}}'
```

## 4. 常见问题

### 4.1 登录问题

- **忘记密码**：点击"忘记密码"链接，按照提示重置密码
- **账号被锁定**：联系管理员解锁

### 4.2 估价结果问题

- **结果不准确**：在估价结果页面点击"反馈"按钮，提供反馈信息
- **需要人工修正**：登录数据后台，手动修正估价结果

### 4.3 报告生成问题

- **报告生成失败**：检查估价记录是否完整，重新生成报告
- **报告格式不符**：联系管理员调整报告模板

## 8. 术语解释

- **置信度**：估价结果的可信度，范围0-1
- **异常数据**：与预期不符的估价数据
- **模型精度**：估价模型的准确程度
- **API**：应用程序编程接口，用于系统间通信
