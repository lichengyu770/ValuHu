数智估价核心引擎 - API 演示

1. 数据库连接测试

此示例演示如何调用 API 测试数据库连接是否正常工作。

2. 从前端调用 API 的示例代码

// 使用 fetch API 调用后端接口
async function fetchData() {
    try {
        const response = await fetch('http://localhost:3003/api/test');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Error:', error);
    }
}

3. 如何在现有页面中集成 API

您可以在任何现有的 HTML 页面中添加以下代码来调用 API：

<script>
// 在页面加载完成后调用 API
document.addEventListener('DOMContentLoaded', async function() {
    try {
        const response = await fetch('http://localhost:3003/api/valuation');
        const data = await response.json();
        
        // 将数据显示在页面上
        const container = document.getElementById('valuationData');
        container.innerHTML = JSON.stringify(data, null, 2);
    } catch (error) {
        console.error('获取估价数据失败:', error);
    }
});
</script>
